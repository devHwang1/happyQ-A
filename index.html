📖 사용법 가이드

데이터 추출 방법 (5단계)

  1. 1 NSM10 접속 → 수정요청현황 메뉴에 진입한 후, 브라우저에서 F12 를 눌러 개발자 도구를 엽니다.
  2. 2 NSM 조회 버튼을 클릭하여 데이터를 조회합니다.
  3. 3 개발자 도구 상단 탭에서 Network 탭을 선택하고, 네트워크 요청 목록에서 해당 요청을 클릭한 뒤 Response 탭을 확인합니다.
  4. 4 Response 내용을 Ctrl + A 로 전체 선택한 후 Ctrl + C 로 복사합니다.
  5. 5 이 페이지의 「Network Response JSON」 입력창에 붙여넣고 「문서 생성하기」 버튼을 누르면 끝!

버튼 기능 안내

⚡ 문서 생성하기
붙여넣은 JSON을 파싱하여 Q&A 표를 자동으로 생성합니다. 데이터 입력 후 반드시 먼저 실행하세요.
📋 표 복사 (Excel용)
화면에 보이는 Q&A 표 전체를 클립보드에 복사합니다. 그대로 Excel에 붙여넣어 활용하세요.
🤖 AI 요약용 복사
개인정보·불필요 정보를 제거하고 AI가 읽기 좋은 텍스트 형식으로 변환 후 복사합니다. ChatGPT 등 AI에 바로 붙여넣으세요.
⬇ Markdown 다운로드
Q&A 데이터를 Markdown 표 형식(.md)으로 변환하여 파일로 저장합니다. NotebookLM이나 Obsidian 등에 활용하세요.
